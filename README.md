# Image Fitter

画像を正方形に整える、依存なしのシンプルな Web アプリです。  
バニラ JavaScript と `canvas` だけを使って、背景をぼかした正方形画像を生成します。

## 機能

- 画像ファイルを読み込む
- 正方形の出力サイズを指定する
- サイズ未指定時は、入力画像の長辺を正方形の一辺として使う
- 背景画像を中央配置し、正方形いっぱいに敷き詰める
- 背景画像にぼかしをかける
- ぼかし強度は割合で指定し、出力サイズに応じて内部で px に変換する
- メイン画像を中央に収めて合成する
- メイン画像の表示倍率を調整する
- 出力画像形式を `PNG / JPEG / WebP` から選べる
- `JPEG / WebP` では品質設定を調整できる
- `ライブプレビュー` を有効にすると、設定変更時にリアルタイムで再描画する

## ファイル構成

- `[index.html](./index.html)`  
  画面の構造と入力フォームを定義します。
- `[styles.css](./styles.css)`  
  画面レイアウトと見た目を定義します。
- `[app.js](./app.js)`  
  画像の読み込み、`canvas` への描画、ダウンロード処理を実装しています。

## 使い方

1. `index.html` をブラウザで開きます。
2. 必要に応じて正方形サイズを入力します。
3. ぼかし強度を調整します。
4. 必要に応じてメイン画像の大きさを調整します。
5. 画像ファイルを選択します。
6. 必要に応じて出力画像形式と品質設定を調整します。
7. 次のどちらかでプレビューを更新します。
   - `ライブプレビュー` がオフ: `プレビュー` ボタンを押す
   - `ライブプレビュー` がオン: 入力変更のたびに自動で再描画される
8. `ダウンロード` ボタンから生成画像を保存します。

## 画像の合成ルール

### 背景画像

- 元画像の短辺が正方形サイズに合うように拡大します。
- その結果、長辺側ははみ出すため、中央配置した状態で上下または左右が切り取られます。
- ぼかし強度は UI 上では `%` で指定します。
- 内部では、`ぼかし半径(px) = 正方形サイズ × (ぼかし強度 / 100) × 0.1` として `canvas` の `filter: blur(...)` に変換します。

### メイン画像

- 元画像の長辺が正方形サイズに合うように拡大または縮小します。
- 初期値 `100%` では、画像全体が収まる最大サイズで中央配置されます。
- 表示倍率を変更すると、中央配置を保ったままメイン画像だけを拡大・縮小できます。
- `100%` を超えると画像の一部が正方形の外にはみ出す場合があります。

## 技術メモ

- 外部ライブラリは使用していません。
- 画像加工はすべてブラウザ標準の `canvas` API を使っています。
- 出力形式は `PNG / JPEG / WebP` を選択できます。
- `JPEG / WebP` では `canvas.toDataURL()` の品質値を調整します。

## 補足

- 非常に大きな画像を扱うと、ブラウザのメモリ使用量が増える場合があります。
- `canvas` のぼかし表現はブラウザ実装に依存するため、環境によって見え方がわずかに異なることがあります。
